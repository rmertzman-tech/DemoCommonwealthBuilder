Name the Deed (The Shared Function)

‚ùå BAD:

  • "Make finance moral" (Not testable, assumes one morality)
  • "Force divestment" (Uses conflict language)

  • "Reduce financed-emissions intensity ‚â•5% while maintaining return within ¬±0.5% vs. benchmark"

Development Roadmap

Phase 1: Demo & Concept Validation (Current)

Timeline: Week 1-2

Goals:

  • Create interactive demo (this page!)
  • Present to students and faculty
  • Recruit development team
  • Validate that people understand the concept

Status: In progress. Share feedback!

Phase 2: Core Functionality Build

Timeline: 6-8 weeks

Team Needed: 2-3 frontend developers, 1-2 backend developers, 1 UI/UX designer

Features:

  • Complete all 8 screens of the builder flow
  • Identity Kernel tool
  • Conscience Gate mechanism
  • PDF export with professional formatting
  • User accounts and save progress
  • Mobile-responsive design

Phase 3: Community & Learning Features

Timeline: 8-12 weeks

Team Needed: Add 1 backend dev, 1 content curator, 2-3 testers

Features:

  • Example library (browse successful builds)
  • Active build dashboard (track your proofs)
  • Change-log feature (document learning)
  • Three-lens ethics check (optional tool)
  • Capability matcher (find collaborators)

Phase 4: Beta Test & Iterate

Timeline: 4-6 weeks

Goals:

  • 50+ beta users build real proofs
  • Track: completion rate, launch rate, 60-day sustainability
  • Iterate based on feedback
  • Prepare for public launch
